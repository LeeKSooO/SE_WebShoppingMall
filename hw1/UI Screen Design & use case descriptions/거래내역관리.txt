// 이한슬

// 구매 내역 조회
Actor Action 
[1] None
System Response 
[2] 회원이 구매했던 상품 내역 목록을 상품명을 기준으로 오름차순으로 정렬하여 보여준다.
Actor Action 
[3] 회원이 구매내역 하나를 선택한다.
System Response
[4] 상세 구매 내역(상품명, 제작 회사명, 판매자, 가격, 평균 구매 만족도, 구매일)을 보여준다.

// 구매 만족도 평가
Actor Action 
[1] 구매 내역을 조회한다.
System Response 
[2] 상품 구매 내역을 보여준다.
Actor Action 
[3] 조회된 구매 내역 중 하나의 상품을 선택한다.
System Response
[4] 해당 상품의 상세 구매 내역(상품명, 제작 회사명, 판매자, 가격, 평균 구매 만족도, 구매일)을 보여준다.
    이때, 구매 만족도를 평가할 수 있도록 별점 칸을 제공한다.
Actor Action
[5] 별점 칸에 구매 만족도를 평가한다.
System Response
[6] 회원이 입력한 구매 만족도로 상세 구매 내역에 저장하여 표시한다.

// 환불 신청
Actor Action 
[1] 구매 내역을 조회한다.
System Response 
[2] 상품 구매 내역을 보여준다.
Actor Action 
[3] 조회된 구매 내역 중 하나의 상품을 선택한다.
System Response
[4] 해당 상품의 상세 구매 내역(상품명, 제작 회사명, 판매자, 가격, 평균 구매 만족도, 구매일)을 보여준다.
    이때, 환불 신청 버튼을 제시한다.
Actor Action
[5] ‘환불 신청’ 버튼을 클릭한다.
System Response
[6] 환불 신청 팝업을 띄운다.
Actor Action
[7] 환불 신청 사유와 같은 추가 내용을 입력한 후, ‘신청’ 버튼을 클릭한다.
System Response
[8] 환불 신청이 완료되었다는 확인 메시지 팝업을 띄운다.
