// 이한슬

[1]
Actor: 회원
Use Cases: 구매 내역 조회, 구매 만족도 평가, 환불 신청, 구매 내역 삭제, 판매/구매 상품 총액과 평균 구매 만족도 출력
Description: 회원은 시스템에서 제공하는 구매 내역 조회, 구매 만족도 평가, 환불 신청, 구매 내역 삭제, 판매/구매 상품 총액과 평균 구매 만족도 출력 기능들을 사용할 수 있다.

[2]
Actor(Event): 환불 신청
Use Cases: 물품 수거 신청
Description: 회원이 어떤 구매 상품에 대하여 환불 신청을 하는 Event가 발생하면 자동으로 물품 수거 신청이 요청된다.

[3]
Actor: 택배사(외부 시스템)
Use Cases: 물품 수거 신청
Description: 회원이 환불 신청을 하는 Event가 일어나면 외부 시스템인 해당 상품을 배송하는 택배사로 물품 수거 신청이 요청된다.

[4]
Actor(Event): 거래 내역 6개월 지남
Use Cases: 구매 내역 삭제
Description: 거래 내역이 6개월이 지나면 자동으로 삭제된다.

[5]
Actor(Event): 매월 말일
Use Cases: 통계 정보 통지
Description: 매월 말일에 모든 회원들에게 그 달에 대한 판매 및 구매 통계 정보가 공지된다.

[6]
Actor: 이메일 시스템(외부 시스템)
Use Cases: 통계 정보 통지
Description: 매월 말일, 회원들에게 통계 정보가 공지될 때, 외부 시스템인 이메일 시스템을 통해 각 회원들에게 공지가 발송된다.
